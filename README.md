# 项目简介 python924

本项目是基于Python的Django框架与Vue.js的前端技术，开发的一套新生报到服务管理系统。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

系统主要包括以下功能模块：

- **首页**：展示系统概览。
- **个人中心**：用户可以管理个人信息。
- **用户管理**：管理系统用户账户。
- **新生信息管理**：维护新生相关信息。
- **宿舍安排管理**：管理新生宿舍分配情况。
- **系统管理**：对系统进行配置管理。

## 系统角色

系统主要面向以下角色：

- 新生
- 管理员

## 运行环境

- 后端：Python 3.8+，Django 3.x
- 数据库：MySQL 5.7+
- 前端：Vue.js 3.x

## 部署步骤

1. 准备Python运行环境。
2. 安装依赖库。
3. 配置数据库。
4. 运行后端服务。
5. 构建前端项目。
6. 部署至服务器。
